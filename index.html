Pwc Customer Retention Analysis

This project involved tackling customer retention challenges by establishing key performance indicators (KPIs), developing a dashboard using Microsoft Power BI, and presenting actionable recommendations. The dataset utilized was the Customer Churn Retention dataset provided by PwC. Through data preparation in Power Query and modeling in Power BI Desktop, valuable insights were uncovered, such as the significant churn rate among customers on month-to-month contracts and those utilizing Fiber Optic internet services. Furthermore, the analysis identified the 1-20 age group as the most susceptible to churn.

Pwc Diversity Analysis

The task involves defining appropriate KPIs in hiring, promotion, performance, and turnover for the Diversity and Inclusion dataset. A visualization is created in Microsoft Power BI Desktop to present relevant KPIs and metrics to the HR manager. The dataset is transformed and loaded into Power BI Desktop for modeling. Data cleaning is performed in the power query editor, including changing the header row, removing unnecessary columns and rows, and validating data types. The data visualization in Power BI shows insights such as the gender distribution of employees, promotion percentages, regional employee counts, average ratings, and age group distributions among male and female employees.
